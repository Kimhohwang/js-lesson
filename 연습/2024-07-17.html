1. 변수선언
script 내용

let 통장_잔액 = 5000;

통장_잔액 = 통장_잔액 + 10000;

변수선언에는 let, const, var 3가지가 있고 현재는 var를 잘사용하지 않는다.

2. 변수명 규칙
  • 규칙01. 카멜케이스
  • 규칙02. 줄여쓰기 금지
  • 규칙03. 숫자 사용은 조심
  • 규칙04. 불변 상수

규칙 1 : 변수명을 지정할때 첫단어를 제외한 단어의 시작을 대문자로 표기한다.

규칙 2 : 임의로 단어를 줄여쓰는 행위를 자제한다. 기억하기가 힘들고 여러명이 함께하는 프로젝트에서 혼동이 발생한다.

규칙 3 : 변수명 가장앞에는 숫자를 사용하면 에러가 발생해 숫자사용시 조심해야한다.

규칙 4 : 변하지않는 상수에 대한 변수명은 전체 대문자로 표기한다. 예시: const COLOR_WHITE ="#ffffff";

3. 자료형
script 내용 [숫자]

let a = 100;

let b = 3.1214545;

let c = 100 + 3.151545;

let d = 100 + "문자";

a = 숫자로 인식

b = 숫자로 인식

c = 숫자로 인식

d = 문자로 인식

script 내용 [문자]

let str1 = "큰 따옴표 가능";

let str2 = '작은 따옴표 가능';

let str3 = `변수 삽입도 가능 ${a}`;

"" , '' , `` 사용가능하며 `` 안에는 변수를 넣을수있다.
script 내용 [불리안]

let boolean1 = 100 > 1;

let boolean2 = 100 < 1;

불린형(논리 타입)은 true와 false 두 가지 값밖에 없는 자료형입니다.

불린형은 긍정(yes)이나 부정(no)을 나타내는 값을 저장할 때 사용합니다. true는 긍정, false는 부정을 의미합니다.

script 내용 [null]

null 값은 지금까지 소개한 자료형 중 어느 자료형에도 속하지 않는 값입니다.
